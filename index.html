movies['title_name'] = movies['title'].apply(lambda x: x[:re.search('\(\d+\)',x).start()])

movies['released_year'] = movies['year'].apply(lambda x: re.search('\d+',x).group())

Collaborative Filtering Recommendation System¶

In this section, we build a recommendation system based on the movie rating data using a model-based Collaborative filtering approach, Matrix factorization, to predict the rating of the movies a user has not rated yet and to provide list of recommended movies based on those predicted rating.

The idea behind matrix factorization is to learn latent factors associated with users and movies. We can think of the these latent factors as user's preferences and movie characteristics respectively, which are hidden features but we can learn them from rating data given.

Stochastic Gradient Descent¶

We initialize latent factors of users as P vector and latent factors of movies as Q vector To predict rating of user i on movie j

$$ \hat r_{ui}=P_u^TQ_i $$

And we can work this out as a linear regression problem by minimizing cost function to learn parameter vectors P and Q

To learn latent factors P (for users) and Q (for movies) vector, we minimize cost function: $$ \underset{q , p}{min}\sum_{(u,i)\epsilon K }(r_{ui}-p_u^Tq_i)^2+\lambda(\left \| p_u \right \|^2 + \left \| q_i \right \|^2) $$

where $\lambda$ is regularization term (L2 penalty) to prevent overfitting

Following the cost (objective) function above, we can write stochastic gradient descent algorithm as:

  1. Intitalize P and Q vector of length k with random values (k = number of latent features)
  2. For each data point, we calculate gradient and update weight P and Q simultaneously until converged or reach maximum iteration
$$ P_{(u+1)} = P_{(u)} - \gamma [-\sum_{(u,i)\epsilon K }(r_{ui}-p_u^Tq_i)q_i+\lambda p_u] $$$$ Q_{(i+1)} = Q_{(i)} - \gamma [-\sum_{(u,i)\epsilon K }(r_{ui}-p_u^Tq_i)p_u+\lambda q_i] $$

where $ \gamma $ is learning rate or step size to update gradient

Since the error term $ e_{ui} $ = $ (r_{ui} - \hat r_{ui}) $ = $ (r_{ui}-p_u^Tq_i) $, we can rewrite the gradient descent algorithm as:

$$ P_{(u+1)} = P_{(u)} + \gamma (e_{ui}\cdot Q_i-\lambda\cdot P_u) $$$$ Q_{(i+1)} = Q_{(i)} + \gamma (e_{ui}\cdot P_u-\lambda\cdot Q_i) $$
